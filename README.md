# Airbnb Dynamic Pricing Analysis

## Project Overview

This project analyzes Airbnb listings to understand pricing patterns, room type distribution, neighborhood trends, reviews, and availability.

## Tools Used

* Python
* Jupyter Notebook
* Pandas
* Power BI
* GitHub

## Project Deliverables

* Project Report
* Power BI Dashboard
* Jupyter Notebook
* Dataset
* Presentation

## Key Insights

* Entire home/apartment listings have the highest prices.
* Manhattan is the most expensive neighborhood group.
* Private rooms are more affordable.
* Price and reviews have a weak correlation.
* Availability varies across neighborhoods.
